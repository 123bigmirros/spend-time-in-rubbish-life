# 文章主旨
这个项目是我对生命意义的探索和思考，希望能够对他人度过一个充实的人生具有借鉴和指导意义，借此机会也实现我自己的人生目标。

# 背景 
为什么想要开启这个项目呢？因为我想成为一个伟大的人。我曾幻想各种人物形象或者职业作为这个想法的载体，超人，科学家，老师......
但因为各种主观和客观的因素，我与上面的各种“载体”渐行渐远，但我的初心未曾变过。
我选择以项目的形式作为实现目标的方式，我期望该项目的呈现方式是任何包含信息的媒介，传播是易获得的，我希望未来所有人都能参与项目的编辑。

# 编辑守约 
1. 科学性。该项目的任何内容都要小心求证，随时做好推导重来的准备。
2. 易读性。尽可能口语化表达的同时，内容上也要尽可能易于理解。
3. 可获得性。任何人都能轻易获得该项目并使用。
