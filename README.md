# 项目主旨
这个项目是我自己人生的总结和思考，希望能够为他人度过自己的一生起到借鉴的意义，希望自己也能通过这个项目能给自己平庸的人生带来一丝慰藉。

# 背景 
- 为什么想要开启这个项目呢？
小时候总是希望成为一个伟大的人,我曾幻想各种人物形象或者职业作为这个想法的载体，超人，科学家，老师等等。随着长大，我越发清晰的认识世界，认识自己，意识到自己不是世界的中心，也学着接受自己的平庸，我将这个项目视为自己对平庸的抗争。
- 这个项目想干什么？
成体系的教各位如果好好度过自己的一生。
- 项目未来发展的期望
我project作为内容的呈现方式，以github作为保存平台，希望能帮助到需要帮助的人。

# 编辑守约 
1. 科学性。该项目的任何内容都要小心求证，随时做好推导重来的准备。
2. 易读性。尽可能口语化表达的同时，内容上也要尽可能易于理解。
3. 可获得性。任何人都能轻易获得该项目并使用。
